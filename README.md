# Activity Run (ระบบเข้าร่วมกลุ่มวิ่งและกิจกรรม) 🏃‍♂️💨

Activity Run เป็นเว็บแอปพลิเคชันสำหรับจัดการและเข้าร่วมกิจกรรมการวิ่ง หรือคลับการวิ่งต่างๆ โดยผู้ใช้สามารถสร้างกิจกรรม กำหนดสถานที่ และแชทคุยกับสมาชิกในทีมได้แบบเรียลไทม์

## คุณสมบัติเด่น (Features)
- **ระบบสมาชิก:** สมัครสมาชิกและเข้าสู่ระบบด้วย JWT Authentication พร้อมระบบรักษาความปลอดภัยรหัสผ่านด้วย Bcrypt
- **คัดกรองกิจกรรม:** สร้างกิจกรรมการวิ่งใหม่ หรือเลือกเข้าร่วมกิจกรรมที่สนใจ
- **แชทเรียลไทม์:** สื่อสารกับสมาชิกในทีมผ่านระบบ Chat ที่ทำงานด้วย Socket.io
- **แผนที่แบบโต้ตอบ:** แสดงตำแหน่งที่ตั้งของกิจกรรมผ่านแผนที่ Leaflet และรองรับการค้นหาสถานที่
- **การจัดการสถานะ:** ใช้ Redux Toolkit เพื่อการจัดการข้อมูลที่ลื่นไหลในฝั่ง Frontend
- **จัดการรูปภาพ:** อัปโหลดรูปภาพโปรไฟล์หรือกิจกรรมผ่าน Cloudinary

## เทคโนโลยีที่ใช้ (Tech Stack)

### Frontend
- **React 19** พร้อม **Vite**
- **Tailwind CSS** สำหรับการออกแบบ UI ที่สวยงามและ Responsive
- **Redux Toolkit** สำหรับ State Management
- **React Leaflet** สำหรับแผนที่ (Maps)
- **Socket.io Client** สำหรับการสื่อสารแบบเรียลไทม์
- **Axios** สำหรับการเชื่อมต่อ API

### Backend
- **Node.js** & **Express**
- **PostgreSQL** (ใช้ pg เป็น Driver)
- **Socket.io** สำหรับระบบ Chat
- **JWT & Bcryptjs** สำหรับระบบความปลอดภัย
- **Cloudinary** สำหรับจัดการไฟล์สื่อ
